# CFS 足球模拟器 - 游戏修改器

适用于 Steam 游戏 **CFS (Championship Football Simulator)** 的存档修改器。

## 功能

### 快速修改
- 资金修改（快捷按钮：10万 ~ 9.99亿）
- 经理属性编辑（声望、领导力、沟通、运营、战术点、面子值、经验）
- 球队基础属性（财富、声望、球迷数、球场容量）

### 球员编辑
- 按名字搜索 / 按球队筛选 / 自由球员筛选
- 编辑全部技术属性（速度、射门、短传、长传、抢断、头球、力量、敏捷、盘带、体能等 14 项）
- 编辑精神属性（野心、职业素养、怒气控制等 7 项）
- 一键满属性 / 全队满属性

### 球队与球场
- 球场详细编辑（票价、座位、贵宾室、商店、球场质量等）
- 赞助商管理（解锁全部 / 金额翻 10 倍）

### 一键解锁
设施 / 战术 / 训练方案 / 比赛日方案 / 物品 / 合作伙伴 / 留洋目的地 / 巡回赛 / 地块 / 建筑 / 青训设施 / 任务 —— 一键全部解锁

### 青训球员
- 查看全部青训球员列表
- 双击编辑（能力、潜力、成长、纪律）
- 一键全部满潜力

## 使用方法

### 方式一：直接运行 EXE（推荐）
1. 从 [Releases](../../releases) 下载 `CFS修改器.exe`
2. 将 EXE 放到游戏根目录（`steamapps/common/CFS/`）
3. **关闭游戏**
4. 双击运行修改器
5. 选择存档 → 点击「加载」
6. 修改想要的内容
7. 重新启动游戏

### 方式二：从源码运行
```bash
# 需要 Python 3.8+
python cfs_trainer.py
```

### 打包为 EXE
```bash
# Windows
build.bat

# 或手动
pip install pyinstaller
pyinstaller --onefile --windowed --name "CFS修改器" cfs_trainer.py
```

## 技术原理

CFS 使用 SQLite 数据库存储所有游戏存档数据。修改器直接读写 `.db` 存档文件，无需注入游戏进程。

每次加载存档时会自动备份到 `backups/` 目录。

## 数据库结构

游戏存档包含 62 张表，关键表：

| 表名 | 说明 | 关键字段 |
|------|------|---------|
| `Players` | 球员（25000+） | 72 列：属性、身价、薪资、等级等 |
| `Teams` | 球队（855） | 财富、声望、球迷、阵容等 |
| `TeamOtherInfo` | 球队财务 | 现金、储蓄、贷款 |
| `Misc` | 经理/全局数据 | 经验、面子、声望等 |
| `YouthPlayers` | 青训球员 | 能力、潜力、成长 |
| `Facilities` | 设施 | 解锁状态 |
| `TacticsKnowledge` | 战术 | 解锁状态 |
| `Sponsor` | 赞助商 | 金额、解锁 |
| `StadiumInfo` | 球场 | 座位、票价、质量 |

## 注意事项

- **修改前请关闭游戏**，否则修改可能被覆盖
- 修改器会自动备份存档，如需恢复请到 `backups/` 目录查找
- 过度修改可能影响游戏平衡性和体验
